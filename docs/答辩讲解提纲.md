# 答辩讲解提纲

## 1. 为什么选逻辑回归
- 可解释性强：每个特征权重可直接说明对风险的影响方向。
- 训练和部署成本低，适合毕业设计周期。
- 适合输出概率，天然契合“预警分层”场景。

## 2. “智能”体现在哪里
- 自动从业务数据提取特征。
- 自动计算风险概率和分层。
- 自动生成预警原因文案。
- 自动触发重算（登录、作业、考试、GPA、定时任务）。

## 3. 预警闭环在哪里
- 风险预测落库（risk_prediction）
- 预警记录落库（warning_record）
- 教师干预落库（intervention_record）
- 前端看板和详情页可以追踪历史与处置。

## 4. 与普通成绩管理系统的区别
- 不只是录入成绩，而是做“风险预测 + 分层预警 + 干预闭环”。
- 强调过程数据（出勤/提交率）而非仅结果数据（分数）。
- 支持教师按风险优先级进行针对性干预。
